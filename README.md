# Typecho-AdminHitokoto
为博客后台添加一言,~~魔改于Hello Wolrd~~,基于一言API

## 食用方法
1. 法1
  1. 从仓库首页 => code => get zip => 解压
  2. RENAME AdminHitokoto
  3. 配置开始食用吧
2. 法2
  1. 从Releases处获取
  2. 之后同法1

# Attention!:
1. 目录结构应该为/站点目录/usr/plugin/AdminHitokoto/Plugin.php(否则会加载出错)
